## 📂 Path Syllabus & Key Concept Mastery

Below is a breakdown of the core modules I completed and the specific technical concepts I mastered:

### 1. Cyber Security Introduction
* Learned the fundamental differences between **Offensive Security** (Red Teaming) and **Defensive Security** (Blue Teaming).
* Explored basic security methodologies and introductory career paths within the industry.

### 2. Network Fundamentals
* **OSI Model & TCP/IP Suite:** Deep understanding of layers, data encapsulation, and protocol functions.
* **Networking Hardware & Services:** Hands-on labs covering Routers, Switches, Firewalls, **DHCP**, and **DNS**.
* **IP Addressing:** Mastery of IPv4 structure, Subnetting basics, and Public vs. Private IP spaces.
* **Core Protocols:** Analyzing Ping (ICMP) and utilizing tools like `traceroute`.

### 3. How the Web Works
* **Web Architecture:** Understanding how Clients, Servers, and Application Servers interact.
* **HTTP/S Protocol:** Analyzing HTTP Request/Response structures, status codes, and headers.
* **DNS in Action:** How domain resolution works under the hood.

### 4. Linux Fundamentals
* Navigating the Linux filesystem using the Command Line Interface (CLI).
* File manipulation, searching, and filtering commands (`cat`, `grep`, `find`, `less`, `nano`).
* Understanding file permissions (`chmod`, `chown`) and system user management.
* Introduction to shell scripting basics and package management.

### 5. Windows Fundamentals
* Navigating the Windows OS from a system administration perspective.
* Utilizing built-in administrative tools: **Task Manager**, **Service Control Manager**, and **Event Viewer**.
* Introduction to **Active Directory** concepts and Windows file system structure.
